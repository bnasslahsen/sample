### How do I get set up? ###
Preerquisites:
* Java 8  
* Maven 3  
* Git  


clone the project in an empty folder.   
* git clone -b monetary-amount https://github.com/teq-niq/sample.git  
* cd sample  
* mvn clean package  
* java -jar target/sample-0.0.1.jar  
* launch http://localhost:8080/swagger-ui.html in browser
